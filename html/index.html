Linux X10 universal device driver

(aka Project WiSH)

  • x10dev (Version 1.X) for Kernel 2.4.X:  Linux driver for Kernel 2.4 supporting the CM11A, CM17A, SmartHome PowerLink Serial, and SmartHome PowerLink USB transceivers.

  • x10dev-2 (Version 2.X) for Kernel 2.6.7+ and Kernel 2.4:  for Kernel 2.6.7 and higher supporting the CM11A, SmartHome PowerLink Serial, and SmartHome PowerLink USB.  For Kernel 2.4 supporting the CM11A and SmartHome PowerLink Serial.

  • x10web (Java GUI) Version 1.X:  Java client GUI and socket server for both wish Version 1.X and Version 2.X
